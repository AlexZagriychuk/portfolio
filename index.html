I created a workaround solution by adding the "Snippet injection" in the Netlify "Build & deploy" / "Post processing" section. This snippet adds JavaScript code before the end of the "head" section of the "subfolder" website`s index.html file. This code adds the slash at the end of the URL if the "subfolder" website was opened without the trailing slash in the URL.
